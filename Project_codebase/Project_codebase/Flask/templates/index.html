Query Processor

Welcome to the Query Processor application. Follow the instructions below:

  1. Submit a POST request to /process_query endpoint with a JSON payload containing the query and the number of top results (k).
  2. The application will correct the spelling, suggest synonyms, and return the top k results along with the corrected and expanded query.

Examples:

Query: "apple"

k: 3

JSON Payload: {"query": "apple", "k": 3}

Query: "machine learning"

k: 5

JSON Payload: {"query": "machine learning", "k": 5}
